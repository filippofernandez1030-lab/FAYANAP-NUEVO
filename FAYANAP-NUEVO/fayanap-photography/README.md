# Fayanap Photography — Sitio web

## Cómo publicarlo en GitHub Pages

1. Crea un repositorio nuevo en GitHub (por ejemplo `fayanap-web`).
2. Sube TODO el contenido de esta carpeta (index.html y la carpeta img/) a la raíz del repositorio.
3. En GitHub: Settings → Pages → Source: rama `main`, carpeta `/ (root)` → Save.
4. En 1-2 minutos tendrás tu link: https://tu-usuario.github.io/fayanap-web/

## Categorías incluidas (cada una con galería deslizable de varias fotos)
- Retrato (3 fotos)
- Deportivas (6 fotos)
- Programas de TV y Radio (9 fotos)
- Gastronómicas (5 fotos)
- Cumpleaños (3 fotos)
- Actividades empresariales (6 fotos)

Las categorías "Graduación" y "Pre Bodas" se eliminaron de la página porque no había fotos para ellas. Si más adelante tienes fotos de esos temas, dile a Claude que las agregue.
